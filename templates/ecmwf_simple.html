🔧 技术实现

API调用代码 (实际执行的):

import cdsapi c = cdsapi.Client() c.retrieve( 'reanalysis-era5-single-levels', { 'product_type': 'reanalysis', 'variable': '2m_temperature', 'year': '2024', 'month': '01', 'day': '01', 'time': '00:00', 'area': [31.2, 120.3, 30.8, 120.7], 'format': 'grib', # 专业气象格式 }, 'real_data.grib' )

数据读取代码:

import xarray as xr ds = xr.open_dataset('real_data.grib', engine='cfgrib') # 获取2x2网格数据 lats = ds.latitude.values # [31.05, 30.8] lons = ds.longitude.values # [120.3, 120.55] temps = ds['t2m'].values # 2x2温度矩阵 # K转℃ temps_celsius = temps - 273.15
