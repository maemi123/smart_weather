# 优化作物详情标题和历史分析渐变计划

## 任务1: 修改作物详情页面浏览器标题

### 修改内容
**文件**: templates/crop_detail.html

**当前标题**: `智慧天气丨{{ crop.name }}`
**新标题格式**: `智慧天气丨农业气象·{{ crop.name }}详情`

例如：
- 小白菜页面："智慧天气丨农业气象·小白菜详情"
- 水稻页面："智慧天气丨农业气象·水稻详情"

## 任务2: 历史分析模块添加渐变过渡

### 问题描述
当前历史分析模块从白色导航栏直接跳到橙黄色header，视觉上比较突兀。

### 解决方案
在导航栏和header之间添加一个渐变过渡区域，或者调整header的渐变效果使其与导航栏更好地衔接。

### 具体修改
**文件**: 
- templates/history_yearly.html
- templates/history_trend.html

**修改内容**:
1. 调整 `.page-header` 的渐变背景，使其从顶部白色逐渐过渡到橙黄色
2. 或者添加一个过渡区域，让颜色变化更加平滑

**CSS修改方案**:
```css
.theme-history .page-header {
    /* 从顶部白色渐变到橙黄色 */
    background: linear-gradient(180deg, 
        rgba(255, 255, 255, 1) 0%, 
        rgba(255, 255, 255, 0.9) 10%,
        rgba(255, 143, 0, 0.15) 50%,
        rgba(255, 255, 255, 1) 100%);
    border-bottom: 3px solid var(--history-theme);
    padding: 30px 0;
    margin-bottom: 30px;
    margin-top: -20px; /* 向上延伸到导航栏下方 */
    padding-top: 50px; /* 增加顶部内边距 */
}
```

或者采用另一种方案 - 在导航栏下方添加一个渐变条：
```css
/* 在导航栏和header之间添加渐变过渡 */
.navbar-custom::after {
    content: '';
    display: block;
    height: 3px;
    background: linear-gradient(90deg, 
        transparent 0%, 
        var(--history-theme) 50%, 
        transparent 100%);
    opacity: 0.5;
}
```
