# 增加中层和零度层标签间距

## 问题
当前间距仍然不够，0度层文字标签和中层文字仍有重叠。

## 当前代码
```python
y_mid_bottom = y_mid - 0.09
if y_freeze_real > y_mid:
    y_freeze_label = min(y_freeze_label, y_mid - 0.08)  # 间距0.08
else:
    y_freeze_label = max(y_freeze_label, y_mid_bottom + 0.05)  # 间距0.05
```

## 修复方案
增加间距参数：
- 中层上方间距从 0.08 增加到 0.12
- 中层下方间距从 0.05 增加到 0.10

```python
y_mid_bottom = y_mid - 0.09
if y_freeze_real > y_mid:
    y_freeze_label = min(y_freeze_label, y_mid - 0.12)  # 增加间距
else:
    y_freeze_label = max(y_freeze_label, y_mid_bottom + 0.10)  # 增加间距
```
