# 修复中层和零度层文字重叠问题

## 问题分析

**当前代码**:
```python
# 中层位置
y_mid = min(0.72, max(0.45, y_mid_real))  # 限制在 0.45-0.72
# 中层文字占用: y_mid, y_mid-0.03, y_mid-0.06

# 0度层标签位置
y_freeze_label = 0.5  # 固定位置
# 0度层文字占用: y_freeze_label, y_freeze_label-0.03
```

**问题**:
- 中层最低可能到 0.45，文字最低到 0.39
- 0度层标签固定在 0.5
- 当中层位置接近下限时，中层文字(0.39-0.45)和0度层文字(0.47-0.53)可能重叠

## 解决方案

动态计算0度层标签位置，确保与中层文字保持最小间距：

```python
# 计算中层文字的最低位置
y_mid_bottom = y_mid - 0.09  # 中层文字最低位置（包含所有行）

# 动态计算0度层标签位置
# 确保与中层保持至少0.05的间距
y_freeze_label = max(0.35, min(0.55, y_freeze_real))
y_freeze_label = max(y_freeze_label, y_mid_bottom + 0.05)  # 确保在中层下方

# 如果0度层真实高度高于中层，标签也应该在中层上方
if y_freeze_real > y_mid:
    y_freeze_label = min(y_freeze_label, y_mid - 0.05)
```

## 修改内容

1. 移除固定的 `y_freeze_label = 0.5`
2. 根据中层位置动态计算0度层标签位置
3. 确保最小间距为 0.05（约5%的图表高度）
