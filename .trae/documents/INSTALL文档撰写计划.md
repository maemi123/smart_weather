# INSTALL.md 安装部署文档撰写计划

## 文档结构

### 1. 环境要求

* Python 版本：3.8+（推荐 3.10+）

* 操作系统：Windows&#x20;

* seek API open-meteo API

* 磁盘空间：至少 500MB

### 2. 详细安装步骤

* 步骤1：克隆项目

* 步骤2：创建虚拟环境

* 步骤3：激活虚拟环境

* 步骤4：安装依赖

* 步骤5：配置 API 密钥

### 3. 配置文件说明

* .env 文件配置

* DeepSeek API 密钥获取方法

* 可选配置项（不配置 AI 功能也能运行）

### 4. 运行方法

* 开发环境运行：python app.py

* 访问地址：<http://127.0.0.1:5000>

### 5. 常见问题解决

* 网络问题（API 访问失败）

* 依赖安装失败（cfgrib 需要 eccodes）

* API 密钥问题

* 端口占用

* Windows 编码问题

### 6. 验证安装成功

* 访问首页

* 检查各模块功能
