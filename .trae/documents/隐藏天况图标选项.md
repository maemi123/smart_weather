# 隐藏天况图标选项和显示（当开启ML校正时）

## 需求
在 `/advanced-forecast` 页面的72小时精细化预报中，当开启"机器学习误差校正"后：
- 隐藏"天况图标"复选框选项
- 不显示天况图标数据

## 修改计划

### 修改文件
- `templates/advanced_forecast.html`

### 具体修改

#### 1. 给天况图标选项添加ID
```html
<div class="model-checkbox" id="weather_icons_checkbox">
    <input type="checkbox" id="weather_icons" checked onchange="updateDetailedChart()">
    ...
</div>
```

#### 2. 在 toggleMLCorrection 函数中添加隐藏/显示逻辑
```javascript
// 当开启ML校正时
document.getElementById('weather_icons_checkbox').style.display = 'none';
document.getElementById('weather_icons').checked = false;

// 当关闭ML校正时
document.getElementById('weather_icons_checkbox').style.display = 'flex';
document.getElementById('weather_icons').checked = true;
```

#### 3. updateDetailedChart 函数中已有判断
当前代码已经检查 `document.getElementById('weather_icons').checked`，当复选框被取消勾选时不会显示天况图标。
