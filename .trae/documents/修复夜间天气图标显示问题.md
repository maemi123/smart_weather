# 修复夜间天气图标显示问题

## 问题描述

首页实时天气在夜间晴天时仍然显示太阳相关的天气图标（☀️），应该显示月亮相关的图标（🌙）。

## 问题分析

**当前代码** (templates/index.html 第294-304行):

```javascript
function mapWeatherStyle(code) {
    const c = Number(code);
    if (c === 0) return { icon: '☀️', bg: '...' };  // 晴天 - 始终显示太阳
    if ([1,2,3].includes(c)) return { icon: '⛅', bg: '...' };  // 多云
    // ...
}
```

**问题**: 函数只根据天气代码返回图标，没有考虑当前时间是否是夜间。

## 解决方案

修改 `mapWeatherStyle` 函数，添加夜间判断逻辑：

### 夜间图标映射

| 天气代码     | 白天图标 | 夜间图标    |
| -------- | ---- | ------- |
| 0 (晴)    | ☀️   | 🌙      |
| 1 (基本晴)  | 🌤️  | 🌙      |
| 2 (局部多云) | ⛅    | 🌙☁️    |
| 3 (多云)   | ☁️   | ☁️ (不变) |
| 其他       | 保持原样 | 保持原样    |

### 夜间判断逻辑

* 日出时间约 06:00

* 日落时间约 18:00

* 简化判断：6:00-18:00 为白天，其他为夜间

或者使用更精确的方式：从API获取的 `is_day` 字段（如果有的话）

## 实施步骤

1. 修改 `mapWeatherStyle` 函数，添加时间参数或内部判断当前时间
2. 根据是否夜间返回不同的图标
3. 调整夜间背景渐变色

## 修改代码

```javascript
function mapWeatherStyle(code) {
    const c = Number(code);
    const now = new Date();
    const hour = now.getHours();
    const isNight = hour < 6 || hour >= 18;  // 18:00-06:00 为夜间
    
    // 夜间图标映射
    if (c === 0) {
        return { 
            icon: isNight ? '🌙' : '☀️', 
            bg: isNight 
                ? 'linear-gradient(135deg, rgba(25, 25, 112, 0.28), rgba(255,255,255,1))'  // 深蓝色夜间背景
                : 'linear-gradient(135deg, rgba(255,215,0,0.28), rgba(255,255,255,1))'
        };
    }
    if ([1,2,3].includes(c)) {
        return { 
            icon: isNight ? '🌙⛅' : '⛅', 
            bg: 'linear-gradient(135deg, rgba(135,206,235,0.28), rgba(255,255,255,1))'
        };
    }
    // 其他天气代码保持不变...
}
```
