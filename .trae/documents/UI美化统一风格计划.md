# UI美化统一风格计划

## 项目概述

对Flask天气分析系统的5个主要页面进行统一风格的UI美化，保持原有功能不变，只优化前端样式。

## 设计规范

### 配色方案

* **主色调**: 科技蓝 `#1E88E5`

* **背景色**: 浅灰 `#F5F7FA`

* **卡片背景**: 白色 `#FFFFFF`

* **文字主色**: 深灰 `#2C3E50`

* **文字次色**: 中灰 `#6B7280`

* **边框色**: 浅灰 `#E5E7EB`

### 模块主题色

| 模块    | 主题色       | 用途        |
| ----- | --------- | --------- |
| 首页    | `#1E88E5` | 科技蓝，主色调   |
| 多模式预报 | `#42A5F5` | 浅蓝色，突出图表  |
| 历史气候  | `#FF8F00` | 橙黄色，历史沉淀感 |
| 探空分析  | `#5E35B1` | 深紫色，科技感   |
| 农业气象  | `#43A047` | 绿色，农业主题   |

### 组件规范

* **卡片圆角**: 12px

* **卡片阴影**: `0 4px 20px rgba(0,0,0,0.08)`

* **按钮圆角**: 8px

* **导航栏高度**: 60px

* **容器最大宽度**: 1400px

* **字体**: 系统默认无衬线体（-apple-system, BlinkMacSystemFont, 'Segoe UI'）

### 导航栏规范

* 统一使用白色背景导航栏

* 当前页面高亮显示（底部边框或背景色）

* 浏览器标题统一格式：`智慧天气丨模块名称`

## 页面改造清单

### 1. 首页 (`/`)

**当前状态**: 已有较好基础，使用Bootstrap
**改造内容**:

* 统一导航栏样式（白色背景+主色调点缀）

* 天气卡片优化阴影和圆角

* 导航卡片统一风格

* 标题改为：`智慧天气`

### 2. 多模式预报 (`/advanced-forecast`)

**当前状态**: 自定义样式，需要统一
**改造内容**:

* 替换导航栏为统一风格

* 标签页按钮美化（圆角、阴影）

* 图表容器统一卡片样式

* 数据卡片统一风格

* 主题色使用浅蓝色 `#42A5F5`

* 标题改为：`智慧天气丨多模式预报`

### 3. 历史气候 (`/history/yearly/`)

**当前状态**: 使用Bootstrap+自定义CSS
**改造内容**:

* 统一导航栏

* 概览卡片美化（橙黄色主题）

* 图表区域统一卡片样式

* 控制面板美化

* 标题改为：`智慧天气丨历史气候`

### 4. 探空分析 (`/upperair`)

**当前状态**: 使用Bootstrap，深色导航栏
**改造内容**:

* 导航栏改为统一白色风格

* 控制面板卡片化

* 图表区域添加卡片容器

* 使用深紫色 `#5E35B1` 作为主题点缀

* 标题改为：`智慧天气丨探空分析`

### 5. 农业气象 (`/agro-dashboard`)

**当前状态**: 已有绿色导航栏，基础较好
**改造内容**:

* 微调导航栏为统一风格（保留绿色主题）

* 卡片统一圆角和阴影

* 预警区域美化

* 标题改为：`智慧天气丨农业气象`

## 实施步骤

1. **创建公共CSS文件** (`static/css/common.css`)

   * 定义CSS变量（配色、圆角、阴影）

   * 统一导航栏样式

   * 统一卡片样式

   * 统一按钮样式

2. **逐个页面改造**

   * 引入公共CSS

   * 替换导航栏

   * 美化卡片和容器

   * 更新页面标题

3. **验证测试**

   * 检查各页面显示效果

   * 确保响应式正常

   * 确认功能未受影响
