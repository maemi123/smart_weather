# 改进72小时精细化图表 - 背景色白灰交替

## 需求
将黑色竖线分隔改为不同天的背景颜色白灰白交替显示，使图表更美观且不突兀。

## 实现方案

### 方案：使用 Plotly Shapes 绘制背景色块

Plotly 可以使用 `shapes` 中的 `rect` 类型来绘制矩形背景色块，实现不同天的背景色交替。

### 具体实现

**文件**: [advanced_forecast.html](file:///d:/pythonProject/smart_weather/templates/advanced_forecast.html)

#### 步骤1：修改每天边界识别逻辑

需要识别每天的起点和终点，而不是只识别分隔点：

```javascript
// 识别每天的边界（起点和终点）
const dayRanges = [];
let currentDate = '';
let dayStart = '';
let dayIndex = 0;

labels.forEach((label, index) => {
    const date = label.split(' ')[0];
    if (date !== currentDate) {
        if (currentDate !== '') {
            // 保存前一天的结束点
            dayRanges[dayRanges.length - 1].end = labels[index - 1];
        }
        // 开始新的一天
        dayStart = label;
        dayRanges.push({
            start: dayStart,
            end: '',  // 暂时为空，等下一天开始时填充
            index: dayIndex
        });
        currentDate = date;
        dayIndex++;
    }
});
// 填充最后一天的结束点
if (dayRanges.length > 0) {
    dayRanges[dayRanges.length - 1].end = labels[labels.length - 1];
}
```

#### 步骤2：将黑色竖线改为背景色块

替换原来的 `shapes` 配置，使用 `rect` 类型绘制背景色块：

```javascript
// 添加每天背景色交替
shapes: dayRanges.map((dayRange, idx) => ({
    type: 'rect',
    x0: dayRange.start,
    x1: dayRange.end,
    y0: 0,
    y1: 1,
    yref: 'paper',
    fillcolor: idx % 2 === 0 ? '#ffffff' : '#f5f5f5',  // 白灰交替
    line: {
        width: 0  // 不显示边框
    },
    layer: 'below'  // 放在数据层下方
})),
```

### 颜色方案

- **白色 (`#ffffff`)**: 第1天、第3天...
- **浅灰色 (`#f5f5f5`)**: 第2天、第4天...

这种浅灰色非常柔和，不会干扰数据阅读，同时能清晰区分不同天。

### 备选颜色方案

如果需要更明显的区分，可以使用：
- 白色 (`#ffffff`) 和 更浅的灰色 (`#fafafa`)
- 或者白色 (`#ffffff`) 和 极浅蓝色 (`#f0f8ff`)
