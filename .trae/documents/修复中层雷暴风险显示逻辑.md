# 修复中层"雷暴大风风险"显示逻辑问题

## 问题描述

探空分析页面的"卡通直观图"中，中层显示"雷暴大风风险增高"的逻辑不正确。

## 问题代码分析

**文件**: sounding\_plotter.py 第285-296行

```python
is_dry = mid_depression > 15
...
risk_icon = "⚡" if is_dry else "☁️"
risk_text = "雷暴大风风险增高" if is_dry else "云层较厚"
```

**当前逻辑**:

* 中层干燥（温露差 > 15°C）→ 显示"雷暴大风风险增高"

* 中层湿润 → 显示"云层较厚"

**问题**:
这个逻辑是**错误**的！

## 正确的气象逻辑

### 中层干燥（温露差大）

* 表示空气干燥，不利于云层发展

* 可能是下沉气流主导

* **不应该**直接判断为"雷暴大风风险增高"

### 中层湿润（温露差小）

* 表示空气接近饱和，有云层形成条件

* 配合其他条件才可能有雷暴风险

### 雷暴大风风险的正确判断

应该基于综合指标：

* CAPE值高

* K指数高

* 风切变大

* 低层湿润 + 中层干燥（干侵入）的组合

## 修复方案

### 方案1: 简化修复（推荐）

修改中层风险提示为更准确的描述：

```python
# 中层干燥
if is_dry:
    risk_icon = "☀️"
    risk_text = "空气干燥，云层较少"
else:
    risk_icon = "☁️"
    risk_text = "湿度较高，云层可能发展"
```

### 方案2: 综合判断

如果需要显示雷暴风险，应该基于CAPE等综合指标：

```python
# 在函数参数中传入cape等指标
# 根据cape判断雷暴风险
if cape > 1500:
    risk_icon = "⚡"
    risk_text = "对流能量充足，注意雷暴风险"
elif is_dry:
    risk_icon = "☀️"
    risk_text = "空气干燥"
else:
    risk_icon = "☁️"
    risk_text = "湿度适中"
```
