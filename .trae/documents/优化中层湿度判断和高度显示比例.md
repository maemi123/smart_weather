# 优化中层湿度判断和高度显示比例

## 问题分析

### 问题1: 中层湿度判断过于"一刀切"

**当前代码** (sounding\_plotter.py 第285-288行):

```python
is_dry = mid_depression > 15
status_color = 'red' if is_dry else 'green'
status_icon = "🤧" if is_dry else "💧"
status_text = "严重干燥" if is_dry else "湿润"
```

**问题**: 只有干燥/湿润两级判断，没有中间过渡。

**对比地面湿度判断**（有三级）:

```python
is_humid = sfc_rh > 90  # 极度潮湿
40 < sfc_rh < 70  # 舒适
其他  # 干燥
```

### 问题2: 高度显示未按正确比例

**当前代码**:

```python
y_high = 0.8  # 高空 (10km+) - 固定位置
y_mid = 0.65  # 中层 (5km) - 固定位置
y_sfc = 0.2   # 地面 - 固定位置
```

**问题**: 高空、中层、地面的标签位置是固定的，没有按真实高度比例显示。

**对比0度层和逆温层**（使用真实高度）:

```python
y_freeze_real = h_to_y(freezing_hght)  # 按真实高度计算Y坐标
```

## 修复方案

### 1. 中层湿度判断改为三级

```python
# 中层湿度判断（三级）
if mid_depression > 15:
    # 干燥
    status_color = 'red'
    status_icon = "🤧"
    status_text = "严重干燥"
elif mid_depression > 8:
    # 适中
    status_color = 'orange'
    status_icon = "😃"
    status_text = "湿度适中"
else:
    # 湿润
    status_color = 'green'
    status_icon = "💧"
    status_text = "湿润"
```

**温露差标准**:

* <br />

  > 15°C: 严重干燥（不利于云层发展）

* 8-15°C: 湿度适中（正常状态）

* < 8°C: 湿润（有利于云层发展）

### 2. 风险提示也相应调整

```python
if mid_depression > 15 and cape > 1500:
    risk_icon = "⚡"
    risk_text = "对流能量充足且温露差大，雷暴大风风险增高"
elif mid_depression > 15:
    risk_icon = "☀️"
    risk_text = "空气干燥，云层较少"
elif mid_depression > 8:
    risk_icon = "⛅"
    risk_text = "湿度适中，天气稳定"
else:
    risk_icon = "☁️"
    risk_text = "湿度较高，云层可能发展"
```

### 3. 高度显示按比例调整

使用 `h_to_y()` 函数计算真实高度对应的Y坐标：

```python
# 高空 (约10km) - 按真实高度
y_high_real = h_to_y(10000)  # 10km
y_high_label = min(0.85, max(0.7, y_high_real))  # 限制在合理范围

# 中层 (约5km) - 按真实高度
y_mid_real = h_to_y(5000)  # 5km
y_mid_label = min(0.7, max(0.5, y_mid_real))  # 限制在合理范围

# 地面 - 固定在底部
y_sfc = 0.2
```
