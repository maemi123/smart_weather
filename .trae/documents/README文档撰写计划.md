# README.md 文档撰写计划

## 文档结构

### 1. 项目简介

* 名称：智慧天气 - 农业气象服务系统

* 一句话描述：基于Flask的智能天气分析系统，集成ECMWF/ERA5气象数据，为浙江5种典型作物提供积温预测和农事建议

* （这里不对，并不光只是农业，农业只是其中一个模块）

### 2. 核心功能模块（每个模块一句话）

* 🏠 首页：项目概览与模块导航

* 📊 多模式预报：ECMWF等数值预报模型可视化，展示未来7天天气趋势

* 🎈 探空分析：基于怀俄明大学探空数据，分析大气垂直结构

* 🌾 农业气象：支持5种浙江典型作物积温预测与AI农事顾问

* 📜 历史气候：杭州历史气候数据可视化

### 3. 农业气象模块详情（重点）

* 作物积温逻辑：

  * 单季晚稻：4月15日播种，积温2200℃成熟（基点10℃）

  * 西湖龙井：1月1日起算，积温380℃春茶开采（基点10℃）

  * 杨梅：1月1日起算，积温950℃成熟（基点10℃）

  * 柑橘：1月1日起算，积温1800℃成熟（基点12.5℃）

  * 小白菜：用户选择播种日，积温400℃采收（基点5℃）

* AI农事顾问：DeepSeek API

### 4. 技术栈

* 后端：Python 3.8+ + Flask

* 数据处理：Pandas + NumPy + Xarray

* 数据源：Open-Meteo API + ERA5再分析数据 + 怀俄明大学探空数据+RP5历史数据+metaestat API历史数据

* 可视化：Matplotlib + Plotly + ECharts

* AI能力：DeepSeek API

* 机器学习：Scikit-learn（随机森林误差校正）
